Referencias Elementos HTML

Tag Descrição
<audio> A tag é usada para incorporar conteúdo sonoro em um documento, como música ou outros fluxos de áudio.
<basefont> A tag foi usada no HTML 4 para especificar uma família padrão de cor de texto, tamanho de fonte ou fonte para todo o texto em um documento HTML.
<em> A tag é usada para definir texto enfatizado. O conteúdo interno é normalmente exibido em itálico.
<fieldset> A tag é usada para agrupar elementos relacionados em um formulário.
<frame> A tag foi usada em HTML 4 para definir uma janela (quadro) em particular dentro de um <frameset>.
<h1-h6> As tags to são usadas para definir títulos HTML.
<main> A tag especifica o conteúdo principal de um documento.
<optgroup> A tag é usada para agrupar opções relacionadas em um elemento <select> (lista drop-down).
<rp> A tag pode ser usada para fornecer parênteses em torno de um texto ruby, a ser mostrado por navegadores que não suportam anotações de ruby.
<ruby> A tag especifica uma anotação de ruby.
<script> A tag é usada para incorporar um script do lado do cliente (JavaScript).
<section> A tag define uma seção em um documento.
<strong> A tag é usada para definir texto com forte importância. O conteúdo dentro é normalmente exibido em negrito.
<tbody> A tag é usada para agrupar o conteúdo do corpo em uma tabela HTML.
<tfoot> A tag é usada para agrupar o conteúdo do rodapé em uma tabela HTML.
