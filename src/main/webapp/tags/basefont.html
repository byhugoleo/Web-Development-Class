Basefont

Tag não suportada no HTML5

A <basefont> tag foi usada no HTML 4 para especificar uma cor de texto padrão, tamanho de fonte ou família de fontes para todo o texto em um documento HTML.

O que usar em vez disso?

Exemplo

Especifique uma cor de texto padrão para uma página


                    <html>
                      <head>
                        <style>
                          body {
                            color: red;
                          }
                        <style>
                      </head>
                    </html>
