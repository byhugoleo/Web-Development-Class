Tag <fieldset>

Exemplo

Agrupa elementos relacionados em um formulário.


                    <form action="/action_page.php">
                      <fieldset>
                          <legend>Formulário:</legend>
                          <label for="fname"> Nome: </label>
                          <input type="text" id="fname" name="fname"><br><br>
                          <label for="lname"> Sobrenome: </label>
                          <input type="text" id="lname" name="lname"><br><br>
                          <label for="email"> Email: </label>
                          <input type="email" id="email" name="email"><br><br>
                          <label for="birthday"> Aniversário: </label>
                          <input type="date" id="birthday" name="birthday"><br><br>
                          <input type="submit" value="Enviar"><br><br>
                      </fieldset>
                    </form>

Definição e uso


                A tag <fieldset> é usada para agrupar elementos relacionados em um formulário.
                A tag <fieldset> traça uma box em torno dos elementos relacionados.

Dicas e Notas


                Dica: A tag <legend> é usada para definir um subtítulo para o elemento <fieldset.>
