Tag <frame>

Tag não suportada no HTML5

A <frame> tag foi usada no HTML4 para definir uma janela específica (frame) dentro de um <frameset>.

O que usar em vez disso?

Use a tag <iframe>para incorporar outro documento no documento HTML atual: 

<iframe src="https://www.w3schools.com"> / >
