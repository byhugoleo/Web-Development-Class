Tag <h1-h6>

Exemplo

Os seis títulos HTML diferentes:


                    <h1>Este é o título 1</h1>
                    <h2>Este é o título 2</h2>
                    <h3>Este é o título 3</h3>
                    <h4>Este é o título 4</h4>
                    <h5>Este é o título 5</h5>
                    <h6>Este é o título 6</h6>

Definição e uso


                As tags to são usadas para definir títulos HTML. <h1><h6>
                <h1> define o título mais importante. define o título menos importante. <h6>
                
Nota: Use apenas um por página - este deve representar o título/assunto principal para toda a página.
Além disso, não pule os níveis de posição - comece com , depois use , e assim por diante. <h1><h1><h2>
