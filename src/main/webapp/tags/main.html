main

Exemplo

Especifica o conteúdo principal do documento:


                    <main>
                      <h1>Navegadores mais populares<\h1>
                      <p>Chrome, Firefox, e Edge são os navegadores mais populares atualmente.<\p>
                     
                    <article>
                      <h2>Google Chrome<\h1>
                      <p>Google Chrome é um navegador desenvolvido pela Google, lançado em 2008.<\p>
                    </article>
                     
                    <article>
                      <h2>Mozilla Firefox<\h1>
                      <p>Mozilla Firefox é um navegador open-source desenvolvido pela Mozilla.<\p>
                    </article>
                     
                    <article>
                      <h2>Microsoft Edge<\h1>
                      <p>Microsoft Edge é um navegador desenvolvido pela Microsoft, lançado em 2015.<\p>
                    </article>

Definição e uso


                A tag <main>tag é usada para especificar o conteúdo principal do documento.
                O conteúdo dentro do elemento <main> deve ser único no documento.  
                Não deve conter nenhum conteúdo que é repetido em vários documentos como barras laterais, links de navegação, informação de copyright, logos do website e formulários de pesquisa.
                Nota:  Não deve existir mais de um elemento <main> em um documento. Ele não deve ser um descendente de <article>, <aside>, <footer>, <header> ou <nav>.
