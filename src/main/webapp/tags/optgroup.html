Optgroup

Exemplo

Agrupe opções relacionadas com tags <optgroup>:


                    <label for="cars">Choose a car:</label>
                    <select name="cars" id="cars">
                      <optgroup label="Swedish Cars>
                        <option value="volvo">Volvo</option>
                        <option value="saab">Saab</option>
                      </optgroup>
                      <optgroup label="German Cars>
                        <option value="mercedes">Mercedes</option>
                        <option value="audi">Audi</option>
                      </optgroup>
                    </select>

Definição e uso


                A <optgroup>tag é usada para agrupar opções relacionadas em um elemento <select> (lista suspensa).
                Se você tiver uma longa lista de opções, os grupos de opções relacionadas serão mais fáceis de manipular para um usuário.

Atributos

Atributo Valor Descrição
disabled disabled Especifica que um grupo de opções deve ser desabilitado
label text Especifica um rótulo para um grupo de opções
