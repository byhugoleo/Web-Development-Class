Tag <ruby>

Exemplo

Uma anotação de ruby :


                    <ruby>
                      漢 <rt> ㄏㄢˋ </rt>
                    </ruby>

Definição e uso


                A tag especifica uma anotação de ruby. <ruby>
                Uma anotação de rubi é um pequeno texto extra, anexado ao texto principal para indicar a pronúncia ou o significado dos caracteres correspondentes.
Esse tipo de anotação é frequentemente usada em publicações japonesas.
                Use junto com <rt> e <rp>: O elemento consiste em um ou mais caracteres que precisam de uma explicação/pronúncia, e um elemento <rt> que dá essa informação,
e um elemento opcional <rp> que define o que mostrar para navegadores que não suportam anotações de rubi.
