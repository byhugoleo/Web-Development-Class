Tag <script>

Exemplo

Escreva "Olá JavaScript!" com JavaScript:


                    <script>
                    document.getElementById("exemplo").innerHTML = "Olá JavaScript!";
                    </script>

Definição e uso


                A tag é usada para incorporar um script do lado do cliente (JavaScript). <script>
                O elemento contém instruções de script ou aponta para um arquivo de script externo através do atributo src. <script>
                Usos comuns para JavaScript são manipulação de imagem, validação de formulários e alterações dinâmicas de conteúdo.
