tbody

Exemplo

Uma tabela HTML com elementos <thead>, <tbody> e <tfoot>:


                    <table>
                      <thead>
                          <tr>
                              <th>Mês<\th>
                              <th>Orçamento<\th>
                          </tr>
                      </thead>
                      <tbody>
                          <tr>
                              <td>Janeiro<\td>
                              <td>R$100<\td>
                          </tr>
                          <tr>
                              <td>Fevereiro<\td>
                              <td>R$80<\td>
                          </tr>
                      </tbody>
                      <tfoot>
                          <tr>
                              <td>Soma<\td>
                              <td>R$100<\td>
                          </tr>
                      </tfoot>
                    </table>

Definição e uso


                A tag <tbody> é usada para agrupar conteúdo de body em uma tabela HTML.
                O elemento <tbody> é usado em conjunto com os elementos <thead> e <tfoot> para especificar cada parte de uma tabela (body, header, footer).
                Navegadores podem usar estes elementos para permitir a rolagem do corpo da tabela independentemente do cabeçalho(header) e do rodapé(footer).
                Além disso, ao imprimir uma tabela grande que abrange várias páginas, esses elementos podem permitir que o cabeçalho e o rodapé da tabela sejam impressos na parte superior e inferior de cada página.
                Nota:  O elemento <tbody> deve ter uma ou mais tags <tr> encapsuladas.
