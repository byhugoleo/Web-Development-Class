Tag <tfoot>

Exemplo

Um tabela HTML com elementos <thead>, <tbody>, e <tfoot>:


                    <table>
                      <thead>
                          <tr>
                              <th>Mês</th>
                              <th>Economia</th>
                          </tr>
                      </thead>
                      <tbody>
                          <tr>
                              <td>Janeiro</td>
                              <td>R$100</td>
                          </tr>
                          <tr>
                              <td>Fevereiro</td>
                              <td>R$300</td>
                          </tr>
                      </tbody>
                      <tfoot>
                          <tr>
                              <td>Soma</td>
                              <td>R$400</td>
                          </tr>
                      </tfoot>
                    </table>

Definição e uso


                A tag é usada para agrupar conteúdo de rodapé em uma tabela HTML. <tfoot>
                O elemento é usado em conjunto com os elementos <thead> e <tbody> para especificar cada parte de uma mesa (rodapé, cabeçalho, corpo). <tfoot>
                Os navegadores podem usar esses elementos para permitir a rolagem do corpo da tabela independentemente do cabeçalho e do rodapé.
Além disso, ao imprimir uma grande tabela que abrange várias páginas, esses elementos podem permitir que o cabeçalho e o rodapé
da mesa sejam impressos na parte superior e inferior de cada página.
                
Nota: O elemento deve ter uma ou mais tags <tr> dentro. <tfoot>
